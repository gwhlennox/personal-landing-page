# George Lennox — Personal Portfolio

Personal landing page for George Lennox, Chief of Staff & Customer Success Leader.

**Live:** [personal-landing-page-lemon.vercel.app](https://personal-landing-page-lemon.vercel.app)

## Stack

- [Astro](https://astro.build) — static site generator
- Plain CSS — no framework dependencies
- Deployed on [Vercel](https://vercel.com) via GitHub integration

## Local development

```bash
npm install
npm run dev
```

## Deployment

Pushes to `main` deploy automatically via Vercel.
